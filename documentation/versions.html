Scheherazade New - Versions and Changes

Announcement list

If you would like stay informed of updates to Scheherazade New and other SIL fonts, please subscribe to the SIL Font News Announcement List. For more information see About.

Current versions

The latest version of the font is always available from the Scheherazade New download page as a .zip archive for all major platforms.

Scheherazade New is also available through the TypeTuner Web service, which allows you to choose among the smart font features and download a font with those features preset. This enables them to work in many applications that do not make use of Graphite or OpenType Character Variants.

Previous versions

Previous versions remain available from our Previous Versions archive.
