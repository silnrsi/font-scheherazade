sources/SOURCES.txt
==================

font family. This information should be distributed along with the Scheherazade
fonts and any derivative works.

details including an FAQ.

Source files and folders included:

buildcomp/
    Code for building composites: See the buildcomp/README.txt for details

graphite/
    Graphite code: See the graphite/README.txt for details

opentype/
    OpenType code: See the opentype/README.txt for details

typetuner/
    TypeTuner code

masters/
    Font sources in Unified Font Object format

ScheherazadeNew.designspace
    DesignSpace document used to open the UFOs in GlyphsApp

feature_map.csv
    Mapping from css feature and language tags to TypeTuner features and values

glyph_data.csv
    Complete list of glyphs (and metadata regarding them) for the project

ScheherazadeNew-WOFF-metadata.xml
    Autogenerated metadata for WOFF fonts

classes.xml
    Glyph classes used in by the Graphite and OpenType logic but which
    cannot be algorithmically generated.

jstf.ttx
    Code to add a JSTF table to the font
